File Uploader - Usage Examples

FileUploader uses a grouped options format for better organization and readability. Options are organized into logical categories like urls, limits, mediaCapture, etc.

Code Examples

Basic Configuration

const uploader = new FileUploader('#my-uploader', {
  // URL Configuration
  urls: {
    uploadUrl: '/api/upload',
    deleteUrl: '/api/delete',
  },

  // File Size Limits
  limits: {
    perFileMaxSize: 10 * 1024 * 1024, // 10MB
    totalMaxSize: 50 * 1024 * 1024,  // 50MB
    maxFiles: 5,
  },
});

Full Featured Configuration

const uploader = new FileUploader('#my-uploader', {
  // URL Configuration
  urls: {
    uploadUrl: '/api/upload',
    deleteUrl: '/api/delete',
    downloadAllUrl: '/api/download-all',
  },

  // File Size Limits
  limits: {
    perFileMaxSize: 25 * 1024 * 1024, // 25MB
    totalMaxSize: 100 * 1024 * 1024, // 100MB
    maxFiles: 10,
  },

  // Upload Behavior
  behavior: {
    multiple: true,
    preventDuplicates: true,
    confirmBeforeDelete: true,
  },

  // Media Capture
  mediaCapture: {
    enableFullPageCapture: true,
    enableRegionCapture: true,
    enableScreenCapture: true,
    enableVideoRecording: true,
    enableAudioRecording: true,
    maxVideoRecordingDuration: 120,
  },

  // Carousel Preview
  carousel: {
    enableCarouselPreview: true,
    carouselShowDownloadButton: true,
  },

  // Callbacks
  callbacks: {
    onUploadSuccess: (file, response) => {
      console.log('Uploaded:', file.name);
    },
    onUploadError: (file, error) => {
      console.error('Error:', error);
    },
  },
});

Available Option Groups

// Access default options:
const defaults = FileUploader.getDefaultOptions();

/**
 * Option Groups:
 *
 * urls          - uploadUrl, deleteUrl, downloadAllUrl, configUrl, uploadDir
 * limits        - perFileMaxSize, totalMaxSize, maxFiles
 * perTypeLimits - perFileMaxSizePerType, perTypeMaxTotalSize, perTypeMaxFileCount
 * fileTypes     - allowedExtensions, imageExtensions, videoExtensions, etc.
 * behavior      - multiple, autoFetchConfig, preventDuplicates, cleanupOnUnload
 * limitsDisplay - showLimits, showProgressBar, defaultLimitsView
 * alerts        - alertAnimation, alertDuration
 * buttons       - showDownloadAllButton, showClearAllButton
 * mediaCapture  - enableScreenCapture, enableVideoRecording, enableAudioRecording
 * carousel      - enableCarouselPreview, carouselAutoPreload
 * dragDrop      - enableCrossUploaderDrag, externalDropZone
 * callbacks     - onUploadSuccess, onUploadError, onDeleteSuccess
 */
